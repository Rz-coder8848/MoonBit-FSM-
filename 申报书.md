# 2026 CCF × MoonBit 国产基础软件开源大赛 - 项目申报书

**1. 项目名称**
MoonBit FSM 状态机框架

**2. 项目简介**
本项目为 MoonBit 生态系统提供了一个轻量级、类型安全且高可扩展的有限状态机 (Finite State Machine, FSM) 核心引擎。通过提供声明式的 Builder API，开发者可以极其优雅、直观地配置状态流转、生命周期回调及前置守卫逻辑。

**3. 项目方向与适用场景**
**方向**：生态库建设与工具链基础设施。
**适用场景**：
- WebAssembly 游戏开发中的 NPC AI 逻辑与关卡流程流转。
- Web 前端框架底层复杂的 UI 状态与动画流程控制。
- 复杂业务系统与网络协议栈的工作流和握手解析。

**4. 拟实现的核心功能**
- **构建者模式 (Builder API)**：基于 MoonBit 泛型安全地声明状态节点和事件流。
- **转移与守卫 (Transitions & Guards)**：支持事件驱动的精准状态流转，并可通过 Guard 条件动态拦截非法语义。
- **生命周期切面 (Lifecycle Hooks)**：完备支持 `on_enter` 与 `on_exit` 动作回调。
- **100% 纯 MoonBit**：零外部依赖，极限压榨 WebAssembly 执行效能。

**5. 是否为原创项目、移植项目或参考已有开源项目**
本项目为**完全原创项目**。核心架构及 API 设计均针对 MoonBit 的类型系统与范式量身定制，仅在设计哲学上借鉴了经典状态机模式，无任何第三方源码移植。
